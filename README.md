## Win11OS

Win11OS kde is a light clean theme for KDE Plasma desktop.

In this repository you'll find:

- Aurorae Themes
- Kvantum Themes
- Plasma Color Schemes
- Plasma Desktop Themes
- Plasma Look-and-Feel Settings

## Installation

```sh
./install.sh
```
## Uninstall

```sh
./uninstall.sh
```
